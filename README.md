# Chrome リモート デスクトップの簡単なやり方｜外出先から自宅PCを操作する方法

外出先から自宅のパソコンを操作したいと思ったことはないだろうか。試してみると設定が複雑で、ポート開放やVPN構築でつまずきがちだ。そんなときは **Chrome リモート デスクトップ（Google Remote Desktop）** を使えば、驚くほど簡単に実現できる。

## なぜChrome リモート デスクトップなのか

- **無料** で利用できる
- 面倒なネットワーク設定（ポート開放・固定IP・VPN）が **不要**
- Googleアカウントがあれば **誰でも数分でセットアップ可能**
- Windows / Mac / Linux に対応し、スマホ（Android / iPhone）からも操作できる

## 事前に用意するもの

- 自宅のパソコン（操作される側／ホスト）
- Googleアカウント
- Google Chrome ブラウザ
- 外出先で使う端末（スマホ・タブレット・別のPC）

---

## セットアップ手順

### ホスト側（自宅PC）の設定

1. 自宅のパソコンで **Google Chrome** を開き、Googleアカウントにログインする。
2. アドレスバーに `remotedesktop.google.com/access` と入力してアクセスする。
3. 「リモート アクセスの設定」欄にある **ダウンロードアイコン** をクリックし、拡張機能「Chrome Remote Desktop」を追加する。
4. 画面の案内に従い、ホスト用アプリ（インストーラー）をダウンロードしてインストールする。
5. このパソコンに表示する **名前** を決めて入力する（例：自宅PC）。
6. 6桁以上の **PIN（暗証番号）** を設定する。これが外部からアクセスする際の鍵になるので、推測されにくい番号にする。
7. 「オンライン」と表示されれば、ホスト側の準備は完了。

### クライアント側（外出先の端末）からの接続

8. 外出先の端末で `remotedesktop.google.com/access` にアクセスする（スマホの場合は「Chrome リモート デスクトップ」アプリでもよい）。
9. **ホストPCと同じGoogleアカウント** でログインする。
10. 一覧に表示された自宅PCの名前をタップ／クリックする。
11. 先ほど設定した **PIN** を入力する。
12. 接続が成功すると、自宅のパソコン画面がそのまま表示され、操作できるようになる。

---

## うまくいかないときのチェックポイント

- 自宅PCの電源が入っていて、スリープになっていないか
- ホストとクライアントで **同じGoogleアカウント** を使っているか
- 自宅PCがインターネットに接続されているか
- PINを間違えていないか

スリープで接続が切れる場合は、自宅PCの電源設定でスリープを「なし」にしておくと安定する。

## こんな人におすすめ

- スマホだけでは完結しない作業を外出先で済ませたい
- 自宅PC内のファイルやアプリに出先からアクセスしたい
- 急な対応で自宅の環境が必要になることがある

使うタイミングは人によって限られるかもしれないが、いざというときの安心感は大きい。

## 次のステップ：外出先から自宅PCの電源を入れる

リモートデスクトップの次に挑戦したいのが、**外出先から自宅PCの電源を入れる（Wake on LAN / WoL）** 設定だ。これができれば、自宅PCをスリープや電源オフにしていても出先から起動できる。難易度は高いが、別記事で詳しく解説する予定。
